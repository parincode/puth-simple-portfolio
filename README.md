# Parin Puth | Data Analyst Portfolio

A modern, responsive portfolio website designed for a Data Analyst, featuring a clean dark aesthetic, interactive elements, and a focus on data storytelling.

## 🚀 Live Demo
Once deployed via GitHub Pages, the site will be available at:
[https://parincode.github.io/puth-simple-portfolio/](https://parincode.github.io/puth-simple-portfolio/)

## ✨ Features
- **Modern Dark Theme:** Default high-contrast dark mode inspired by GitHub's aesthetic.
- **Light/Dark Toggle:** Smooth theme switching with state persistence (via `localStorage`).
- **Responsive Design:** Optimized for mobile, tablet, and desktop viewing.
- **Interactive Animations:** Scroll-triggered fade-in effects for project cards and smooth scrolling.
- **Data-Centric Typography:** Uses **JetBrains Mono** for a professional, technical feel.

## 🛠️ Tech Stack
- **HTML5:** Semantic structure.
- **CSS3:** Custom properties (variables), Flexbox, and Grid for layout.
- **JavaScript:** Vanilla JS for interaction and theme management.
- **Typography:** Google Fonts (JetBrains Mono).

## 📂 Project Structure
- `index.html`: Main structural definition.
- `style.css`: Theme variables, responsive layouts, and animations.
- `script.js`: Theme toggling logic and scroll observers.
- `assets/`: Directory for project images and media.

## 🔧 How to Use
1. **Clone the repository:**
   ```bash
   git clone https://github.com/parincode/puth-simple-portfolio.git
   ```
2. **Open in Browser:**
   Simply open `index.html` in any modern web browser.
3. **Local Development:**
   It is recommended to use the "Live Server" extension in VS Code for real-time previewing of changes.
